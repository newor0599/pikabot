Pikabot

Here are some notes for you to control pikabot

Get started

Before doing anything, you need to import cytron's own pikabot library

press CTRL+SHIFT+I, search for Cytron_pikabot and click install. In your file, paste the code below

#include <Cytron_PikaBot.h>

Motor

Setup

Base myMotor(9,3,11,10);

the number are the port for the motor

Element for class Bass

myMotor.powerBase(left_speed,right_speed);

myMotor.moveForward(speed);

myMotor.moveBackward(speed);

myMotor.turnRight(speed);

myMotor.turnLeft(speed);

myMotor.stop();

Ultrasonic sensor

Setup

Ultrasonic myUltrasonic(echo,trigger);

The default port for these are 4,5

Element for class Ultrasonic;

myUltrasonic.getDistance();

Button

Setup

Button myButton(pin);

default port number is 2

Element for class Button

myButton.isPressed();

IRLine sensor

Setup

IRLine myIRLine(pin);

left IRLine sensor is A0 and the right IRLine sensor is A1

Element for IRLine sensor

myIRLine.onLine();
